About How to Use

How to Use HerdKnowledge

HerdKnowledge lets you ask questions about Colorado Parks and Wildlife hunting data in plain English. No more scanning through PDFs on government websites—just ask what you need to know and get instant answers about harvest and population data.

Data Source: This database contains Colorado Parks and Wildlife bow hunting harvest and population data for elk, deer, and pronghorn from 2019-2024.

Available Datasets

Population Data

The population table contains post-hunt population estimates and demographic information.

Available Fields:

  • state - State code (e.g., 'co' for Colorado)
  • species - Species name ('elk', 'deer', or 'pronghorn')
  • herd_name - Specific herd identifier
  • post_hunt_estimate - Population count after hunting season
  • male_female_ratio - Gender ratio in the population
  • year - Year of the data (2019-2024)
  • unit - Game management unit number

Harvest Data

The harvest table contains detailed harvest statistics including success rates and hunter participation.

Available Fields:

  • state - State code (e.g., 'co' for Colorado)
  • species - Species name ('elk', 'deer', or 'pronghorn')
  • year - Year of the data (2019-2024)
  • season - Hunting season identifier
  • unit - Game management unit number
  • adult_male - Number of adult males harvested
  • adult_female - Number of adult females harvested
  • young - Number of young animals harvested
  • total_harvest - Total animals harvested
  • total_hunters - Number of hunters
  • percent_success - Success rate percentage
  • total_rec_days - Total recreation days

Getting the Most from Your Queries

To get the most accurate and useful results, be specific in your questions by including the following filters:

State

Use "Colorado" or "CO" (data is primarily Colorado)

Species

Specify "elk", "deer", or "pronghorn"

Year

Use years 2019-2024

Season

Specify hunting season (for harvest data)

Unit

Game management unit number (e.g., unit 10, unit 20)

How It Works

We've pulled in all the available bow season harvest and population PDFs for elk, deer, and pronghorn from Colorado Parks and Wildlife into our database. Using natural language processing, we've made all that data easy to search. No more scanning through hundreds of pages on government websites.

Simply ask your questions in plain English and get instant answers. The system automatically:

  • Understands your questions and finds the right data
  • Filters results by species, year, state, and unit when you specify them
  • Returns clear, organized data tables
  • Limits results to keep responses manageable
